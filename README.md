### 🎹 KNN + OpenCV 기반 가상 키보드 ( Python)

- 손가락 위치를 OpenCV로 추적하고, 좌표 데이터를 KNN 알고리즘으로 분류하여 키보드 입력을 구현

- 머신러닝과 영상처리의 융합을 학습하고, 사용자 입력을 비접촉식으로 처리하는 방법 고안
